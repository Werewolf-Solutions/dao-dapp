# DAO dapp

`git clone`

`npm i`

Make sure dao-dapp is in same parent folder of dao-project and you deployed smart contracts, in dao-project/ run `make deploy-local`

```
parent_folder/
  dao-dapp
  dao-project
```

`npm start`
